1. บทนำ (Introduction)

โปรเจค SQLite Notes เป็นแอปพลิเคชันจดบันทึก (Notes App) ที่พัฒนาด้วย React Native ร่วมกับ Expo Framework โดยใช้ expo-sqlite เป็น SDK หลักในการจัดเก็บข้อมูลบันทึกลงในฐานข้อมูล SQLite ภายในเครื่องของผู้ใช้

แอปนี้ออกแบบมาเพื่อให้ผู้ใช้สามารถสร้าง อ่าน แก้ไข และลบบันทึก (CRUD Operations) ได้อย่างสะดวก พร้อมฟีเจอร์เสริมอย่างการค้นหา จัดหมวดหมู่ เลือกสี และปักหมุดบันทึกสำคัญ

2. เกี่ยวกับ Expo SQLite

expo-sqlite เป็นไลบรารีจาก Expo SDK ที่ให้สิทธิ์เข้าถึงฐานข้อมูล SQLite ผ่าน JavaScript API โดยข้อมูลจะถูกจัดเก็บในไฟล์ภายในอุปกรณ์ (Local Storage) ทำให้แอปสามารถทำงานแบบ Offline ได้อย่างสมบูรณ์

คุณสมบัติหลักของ expo-sqlite:
  • รองรับ Asynchronous API สำหรับการ query ข้อมูล
  • ข้อมูลคงอยู่ถาวรแม้ปิดแอปหรือรีสตาร์ทเครื่อง
  • รองรับ Prepared Statements เพื่อป้องกัน SQL Injection
  • ใช้งานได้บน Android, iOS, macOS, tvOS, และ Web
  • รวมอยู่ใน Expo Go สามารถทดสอบได้ทันที

SQL สำหรับสร้างตาราง:

CREATE TABLE IF NOT EXISTS notes (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  title TEXT NOT NULL,
  content TEXT NOT NULL,
  category TEXT DEFAULT 'General',
  color TEXT DEFAULT '#6C63FF',
  is_pinned INTEGER DEFAULT 0,
  created_at TEXT DEFAULT (datetime('now','localtime')),
  updated_at TEXT DEFAULT (datetime('now','localtime'))
);

6. วิธีการติดตั้งและใช้งาน

6.1 ข้อกำหนดเบื้องต้น

  • Node.js เวอร์ชัน 18 ขึ้นไป
  • npm หรือ yarn
  • Expo CLI (npm install -g expo-cli)
  • Expo Go App (สำหรับทดสอบบนอุปกรณ์จริง)

6.2 ขั้นตอนการติดตั้ง

# 1. สร้างโปรเจค Expo ใหม่
npx create-expo-app expo-sqlite-notes --template blank

# 2. เข้าไปในโฟลเดอร์โปรเจค
cd expo-sqlite-notes

# 3. ติดตั้ง expo-sqlite
npx expo install expo-sqlite

# 4. รันแอป
npx expo start

# ติดตั้ง EAS CLI
npm install -g eas-cli

# ล็อกอิน Expo
eas login

# Build APK
eas build -p android --profile preview

7. ตัวอย่างโค้ดหลัก

7.1 การเชื่อมต่อ Database

import * as SQLite from 'expo-sqlite';

export async function openDatabase() {
  const db = await SQLite.openDatabaseAsync('notes.db');
  await db.execAsync(`
    CREATE TABLE IF NOT EXISTS notes (
      id INTEGER PRIMARY KEY AUTOINCREMENT,
      title TEXT NOT NULL,
      content TEXT NOT NULL
    );
  `);
  return db;
}

7.2 การเพิ่มบันทึก (Create)

export async function addNote(title, content, category, color) {
  const db = await openDatabase();
  const result = await db.runAsync(
    'INSERT INTO notes (title, content, category, color) VALUES (?, ?, ?, ?)',
    [title, content, category, color]
  );
  return result.lastInsertRowId;
}

7.3 การอ่านบันทึก (Read)

export async function getAllNotes() {
  const db = await openDatabase();
  return await db.getAllAsync(
    'SELECT * FROM notes ORDER BY is_pinned DESC, updated_at DESC'
  );
}

7.4 การแก้ไขบันทึก (Update)

export async function updateNote(id, title, content, category, color) {
  const db = await openDatabase();
  await db.runAsync(
    `UPDATE notes SET title=?, content=?, category=?, color=?,
     updated_at=datetime('now','localtime') WHERE id=?`,
    [title, content, category, color, id]
  );
}

7.5 การลบบันทึก (Delete)

export async function deleteNote(id) {
  const db = await openDatabase();
  await db.runAsync('DELETE FROM notes WHERE id = ?', [id]);
}

9. สรุป

โปรเจค SQLite Notes แสดงให้เห็นถึงการใช้งาน expo-sqlite ในการพัฒนาแอปพลิเคชันมือถือที่มีการจัดการฐานข้อมูลภายในเครื่อง ผู้ใช้สามารถจดบันทึก ค้นหา จัดหมวดหมู่ และจัดการข้อมูลได้ครบถ้วนผ่าน CRUD Operations

ข้อดีของการใช้ expo-sqlite คือ ข้อมูลถูกจัดเก็บในเครื่องทำให้แอปทำงานได้แบบ Offline โดยไม่ต้องเชื่อมต่ออินเทอร์เน็ต เหมาะสำหรับแอปที่ต้องการความเร็วในการเข้าถึงข้อมูลและความเป็นส่วนตัวของผู้ใช้
